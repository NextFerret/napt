# napt (New Advanced Packaging Tool)

napt is an testing package manager (or wrapper?)  for The Arvor Project. it uses chroots to test the packages into  an sandbox. of course that takes time. thats why you have --apply-host
but anyway,
the syntax is:
napt install ← install packages (who would guess?)
napt remove ← remove packages
napt sync ← apt upgrade
napt upgrade ←  obvious again
napt dist-upgrade ← obvious again x2
